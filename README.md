# ArduClock
ArduClock is basically a clock, but its on a Arduino.

This is owned by NmaProjects, redistribution is strictly not allowed.

To setup ArduClock download the latest version of ArduClock, install it (It will autorun the nessecary setup program for Arduino IDE) install Arduino IDE and it will open the file you need to compile/upload to your Arduino click the arrow next to the checkmark, after click the HTML shortcut and click Connect Device, select your Arduino and it will sync the time.
